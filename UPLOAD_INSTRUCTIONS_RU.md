# Bonvanta: установка многоязычной версии

Пакет уже готов для GitHub Pages. В нем есть:

- английская версия: `/`
- немецкая версия: `/de/`
- испанская версия: `/es/`
- французская версия: `/fr/`
- страницы контактов для каждого языка
- языковой переключатель
- `hreflang`, canonical и обновленный `sitemap.xml`
- Google Tag Manager `GTM-5XCBFMQP` на всех страницах
- consent mode: Google Analytics включается только после согласия на аналитические cookies
- английские юридические документы как действующие версии

## Как загрузить

1. Распакуйте ZIP.
2. Откройте GitHub → репозиторий `bonvanta-website`.
3. Нажмите **Add file → Upload files**.
4. Перетащите **все файлы и папки изнутри распакованной папки**.
5. GitHub спросит о совпадающих файлах. Разрешите заменить существующие.
6. Commit message: `Add multilingual Bonvanta website`
7. Нажмите **Commit changes**.
8. Подождите 1–3 минуты и обновите сайт через `Cmd + Shift + R`.

После публикации откройте Google Search Console → Sitemaps и повторно отправьте:

`sitemap.xml`

## Важно о юридических страницах

Юридические документы сохранены на английском как действующие версии. В локализованных футерах ссылки помечены `(EN)`. Это сделано намеренно: юридический перевод лучше публиковать только после отдельной проверки юристом для соответствующей страны.
