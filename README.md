# Bonvanta website

A responsive static website ready for GitHub Pages.

## Included

- Public product landing page
- Free Forever and Growth pricing
- Responsive mobile design
- Terms, Privacy, Refund and Cookie pages
- Cookie consent banner
- Google Analytics consent loader
- GitHub Pages `CNAME`, sitemap, robots file and 404 page

## Before publishing

1. Open `index.html` and replace `G-XXXXXXXXXX` with your Google Analytics 4 Measurement ID. Leave it unchanged if Google Analytics is not ready yet.
2. Review all marketing claims and legal pages one final time.
3. Upload every file and folder in this package to the root of your GitHub repository.
4. Enable GitHub Pages from the `main` branch and root folder.
5. Configure the custom domain as `bonvanta.com` in GitHub Pages.
6. Replace the Namecheap parking/redirect DNS records only after GitHub shows the required DNS configuration.
